# AoWCharOCVMatchingPy
陣形画像から兵士を画像検索する

インプットの陣形画像から兵士を検索し位置を特定する
未来的には位置から７×７の文字列碁盤目データに変換する目的

以下の例の画像は、巨大岩と見習い魔法使いの検出結果を矩形で表現したもの
